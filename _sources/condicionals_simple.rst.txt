##############
Què passa sí…?
##############

"Si tens 18 anys o més llavors ets major d'edat"

De manera gràfica podem representar la condició anterior de la següent manera:


.. graphviz::
    :align: center

    digraph {
        inici[shape="box", style=rounded];
        final[shape="box", style=rounded];
        condicio1 [label="edat >= 18", shape="diamond", style=""];
        instruccio1 [label="escriu «Ets major d'edat»", shape="rectangle", style=""]

        inici-> condicio1;
        condicio1 -> instruccio1 [label="cert"];
        condicio1 -> final[label="fals"];
        instruccio1 -> final;
    }

Fixa't com, depenent de si el nombre és parell, executem una instrucció o no.

En Java ho escriuríem de manera semblant a:

.. code-block:: java

    if (edat >= 18) {
        System.out.println("Ets major d'edat");
    }

La forma bàsica d'aquesta instrucció és:

.. code-block:: java

    if (condició) {
        instruccions;
    }

Allà on diu:

* ``condició`` hi col·locarem qualsevol expressió que retorni un valor
  booleà.

  La condició ha d'aparèixer entre parèntesis ``()``.

  En aquest cas, la condició és que l'edat sigui més gran o igual a 18.  Com
  sabem, l'operador ``>=`` ens retorna cert si el primer operand és més
  gran o igual que el segon.

* ``instruccions`` podem tenir una o més instruccions, com ara
  d'entrada/sortida, declaracions/assignacions, i fins i tot altres
  condicionals!

  Les instruccions han d'aparèixer entre claudàtors ``{}``

  En aquest cas, només tenim una instrucció que ens dirà "Ets major d'edat"

Afegim la instrucció condicional dins d'un programa complet en Java per
poder veure-la dins de context:

.. code-block:: java
    :linenos:

    public class MajorEdat {
        public static void main(String[] args) {
            int edat = Integer.parseInt(args[0]);
            if (edat >= 18) {
                System.out.println("Ets major d'edat");
            }
            System.out.println("Fi del programa");
        }
    }

En aquest programa agafem l'edat del primer argument que ens passen per
línia de comandes. Així, les següents execucions tindran els resultats
indicats:

.. code-block:: console
    :emphasize-lines: 2, 3, 6, 7, 10

    $ java MajorEdat 18
    Ets major d'edat
    Fi del programa

    $ java MajorEdat 19
    Ets major d'edat
    Fi del programa

    $ java MajorEdat 17
    Fi del programa

Fixa't que quan l'edat és major o igual a 18, el programa mostra el
missatge *Ets major d'edat*. Fixa't també en que, independentment de si és
o no major d'edat, el programa acaba executant la línia 7, que segueix  al bloc
d'instruccions del condicional, i mostra el missatge *Fi del programa*.
